Formulario

Ésto es un formulario para llenar con datos. El script hace lo siguiente:

  • El formulario guarda fecha y hora actualizada
  • Datos se almacenan en un objeto
  • Se genera alerta en caso que no se llene un cuadro
  • Se reinicia el formulario una vez enviado los datos
  • Cada objeto se almacena en una lista
  • La lista se guarda en local storage
